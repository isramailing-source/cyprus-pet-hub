## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <YOUR_GIT_URL>
   cd cyprus-pets
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

   ```bash
   npm run dev
   ```
